== About

このソフトウェア（以下GDMocopi）は link:https://www.sony.net/Products/mocopi-dev/jp/[SONY mocopi]の情報を Godot Engineで受信するための**非公式**な GDExtension です。

この GDExtension は Godot Engine 4.2 以上で使用可能です。
